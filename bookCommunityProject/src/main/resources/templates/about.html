What the book community is about

Hello, this page will introduce you to the main functionality of this site.
Initially, it was conceived as a simple addition of books to the database. But who needs an app like this if it doesn't allow users to interact and share their books with each other.
Therefore, this site is a social network for book geeks, it allows users to download books and share them with others by first requesting access permission. Guests cannot download books to their devices, but they can click on the link and read the works locally.
If we talk about the way to download books, then this application provides 2 such ways:
1) The user uploads a link to the resource, and after checking for correctness, he can download the book itself.
2) Either a direct file download is available.
